# Student Finance Management in a University

Note: Change the database password ,name of the database respectively in all the files according to your mysql credentials

1.First source the database.sql file using the command source <location of the file >
Note: Give the location without quotes
2.Then go to the admin folder,Run the Admin_login File using  streamlit run Admin_login.py
3.Then go to the student folder Run the Student_login File using  streamlit run Student_login.py
4.Then go to the vendor folder Run the Vendor_login File using  streamlit run Vendor_login.py
5.Now go the respective links shown in the terminal and work with the websites
6.Make changes to the database using the gui provided and see the changes being reflected in the database

Required packages in python: Streamlit,Streamlit-extras,mysql.connector etc
